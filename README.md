# 🀄 ひらがじゃん Web版 v2

ひらがな牌で単語を作る麻雀風ゲーム「ひらがじゃん」のWebマルチプレイ版。
**点数計算・リーチ・暗カン**に対応した本格派！

## ✨ 機能

- 牌版本格ルール（手牌14枚、2-3-3-3-3 の5単語であがり）
- 2〜4人マルチプレイ（部屋コード招待 / QRコード対応）
- **🎌 リーチ宣言**（テンパイ時、-1000点供託）
- **📣 暗カン**（手札4枚） / **明カン**（他人捨て牌+手札3枚）
- **💰 点数計算**（初期25000点、役満8000点）
- 辞書チェック（Wiktionary / Wikipedia 日本語版API）+ 全プレイヤー投票
- 「次に何をすべきか」を画面上部に常時ガイド表示
- スマホ最適化UI

## 🎯 役一覧

### 通常役
| 役 | 飜 |
|---|---|
| 立直 | 1 |
| 一発 | 1 |
| ツモ / 門前清自摸和 | 1 |
| 嶺上開花 | 1 |
| 海底撈月 / 河底撈魚 | 1 |
| ダブル立直 | 1（+立直と合計2） |
| 清音作り（全単語が清音のみ） | 2 |
| 濁り組み（全単語に濁音類を含む） | 2 |
| 三槓子 | 2 |
| 清音一色（牌種が全て清音） | 3 |

### 役満（8000点）
- 国士無双（あ・か・さ・た・な・は・ま・や・ら・わ・濁音・半濁音・拗音・長音 を全て揃える）
- しりとり（5単語が順番にしりとりになっている）
- 四槓子（カン4回）
- 天和（配牌時にあがり）

通常役の点数 = `1000 × 2^(飜-1)`（上限8000）

## 🚀 起動方法

```bash
npm install
npm start
# ブラウザで http://localhost:3000 にアクセス
```
